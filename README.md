# fi_farm

ラズベリーパイで畑を観察する.


# 使い方

githubからfi_farmリポジトリをダウンロードし、setupスクリプトを　実行する.

```
git clone https://github.com/peace098beat/fi_farm.git ~/fi_farm
cd ~/fi_farm

cp private.org private

# privateを編集する

bash setup.sh
```

以上.


## 仕様

作業ディレクトリ
```
${HOME}/fi_farm
${HOME}/fi_farm/scripts
${HOME}/fi_farm/logs
```
